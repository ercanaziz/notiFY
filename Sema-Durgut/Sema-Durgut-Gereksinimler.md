1. Fiyat Geçmişini Getirme



API Metodu: GET /products/{id}/history



Açıklama: Seçili ürünün geçmiş tarihlerdeki fiyat kayıtlarını liste olarak sunar.





2. Fiyat Grafiği Verisi Sağlama



API Metodu: GET /products/{id}/chart-data



Açıklama: Grafik çizimi için zaman serisi tabanlı fiyat verilerini hazırlar.





3. En Düşük Fiyatı Sorgulama



API Metodu: GET /products/{id}/lowest-price



Açıklama: Ürünün tüm zamanlardaki en düşük fiyat bilgisini kullanıcıya gösterir.





4. Mağaza Karşılaştırması Yapma



API Metodu: GET /products/compare



Açıklama: Aynı ürünün farklı e-ticaret sitelerindeki fiyatlarını yan yana getirir.





5. İndirim Oranı Hesaplama



API Metodu: GET /products/{id}/discount-rate



Açıklama: Ürünün piyasa ortalamasına göre ne kadar karlı olduğunu hesaplar.





6. Renge Göre Filtreleme



API Metodu: GET /products/color



Açıklama: Kullanıcıların ürünleri belirli renk seçeneklerine göre süzerek listelemesi için kullanılır.





7. Fiyat Tahmini İzleme 



API Metodu: GET /products/{id}/forecast



Açıklama: Mevcut trendlere göre fiyatın gelecekteki olası yönünü raporlar. (Yapay zeka kullanımı ile)
